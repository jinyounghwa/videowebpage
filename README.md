# VideoWebpage - 넷플릭스 클론 데모

동적 콘텐츠 필터링, 맞춤형 비디오 플레이어, 반응형 디자인을 갖춘 고충실도 인터랙티브 넷플릭스 클론 프로젝트입니다. 사용자에게 실제 스트리밍 플랫폼의 핵심 경험을 제공하는 것을 목표로 제작되었습니다.

## 🚀 주요 기능

-   **동적 네비게이션**: 홈, 시리즈, 영화, 요즘 대세, 내가 찜한 리스트 간의 전환을 페이지 새로고침 없이 매끄럽게 지원합니다 (SPA 방식).
-   **인터랙티브 UI/UX**:
    -   콘텐츠 카드에 대한 부드러운 호버 애니메이션 및 확대 효과.
    -   스크롤 시 투명도에서 불투명으로 전환되는 반응형 헤더.
    -   클릭 시 확장되는 인터랙티브형 검색창 기능.
-   **통합 비디오 플레이어**:
    -   전체 재생 제어(재생/일시정지, 탐색, 볼륨, 음소거)가 가능한 커스텀 모달 플레이어.
    -   사용자 편의를 위한 포괄적인 키보드 단축키 지원.
    -   실시간 재생 상태 알림 및 진행 상황 추적 기능.
-   **찜하기 시스템**: 사용자가 선호하는 콘텐츠를 실시간으로 북마크하고 관리할 수 있는 세션 기반 시스템.
-   **콘텐츠 검증**: 모든 데이터셋 내 외부 미디어 자산(이미지, 비디오)의 유효성을 자동으로 검사하는 Node.js 유틸리티 포함.

## 🛠 기술 스택

-   **Frontend**: HTML5, CSS3 (Modern Flexbox, CSS Grid, Custom Keyframe Animations).
-   **Logic**: Vanilla JavaScript (ES6+), 모듈형 객체 지향 설계.
-   **Tooling**: Node.js (자동화된 자산 검증 스크립트용).

## 📥 시작하기

### 사전 요구 사항
-   최신 웹 브라우저 (Chrome, Firefox, Edge, Safari 등).
-   Node.js (선택 사항, 링크 검증 기능을 사용할 경우에만 필요).

### 실행 방법
브라우저에서 `index.html` 파일을 열어 바로 실행할 수 있습니다:
```bash
# 또는 브라우저에서 직접 열기
open index.html
```

### 링크 검증 방법
포스터, 배경 이미지, 비디오 샘플 등 모든 콘텐츠 자산이 정상적으로 연결되어 있는지 확인하려면 다음 명령어를 실행하세요:
```bash
node link-validator.js
```

## 📂 프로젝트 구조

```text
├── css/
│   ├── style.css         # 레이아웃, 타이포그래피 및 주요 컴포넌트 스타일
│   └── animations.css    # 인터랙티브 전환, 호버 효과 및 모달 애니메이션
├── js/
│   ├── data.js           # 중앙 집중식 콘텐츠 저장소 (CONTENT_DATA)
│   ├── player.js         # 비디오 플레이어 모듈 및 키보드 컨트롤러
│   └── app.js            # 메인 애플리케이션 오케스트레이터 (네비게이션, 렌더링, 검색)
├── index.html            # 메인 애플리케이션 쉘 및 모달 템플릿
├── link-validator.js     # Node.js 자산 상태 확인 유틸리티
└── README.md             # 프로젝트 문서
```

## ⌨️ 비디오 플레이어 단축키 안내

비디오 플레이어가 활성화된 상태에서 다음 단축키를 사용하여 편리하게 시청할 수 있습니다:

| 키 | 동작 |
| :--- | :--- |
| `Space` / `K` | 재생 / 일시정지 |
| `ArrowRight` | 10초 앞으로 이동 |
| `ArrowLeft` | 10초 뒤로 이동 |
| `ArrowUp` | 볼륨 증가 |
| `ArrowDown` | 볼륨 감소 |
| `M` | 음소거 전환 |
| `F` | 전체화면 전환 |
| `Esc` | 플레이어 닫기 |
